⚽ ¿Qué es?

Multi-Liga Analytics es una herramienta que analiza partidos de fútbol usando modelos estadísticos (distribución de Poisson y Normal CDF) alimentados con datos reales de ESPN. Genera predicciones para cualquier enfrentamiento: marcadores, goles, tarjetas, corners y más.

Motor de Análisis
  • footballcsv — Resultados históricos de Liga MX (CSV de GitHub)
  • ESPN API — Partidos recientes + estadísticas detalladas (posesión, tiros, corners, tarjetas, faltas, offsides, atajadas)
  • Scraper — Combina fuentes, deduplica, enriquece y genera archivos de datos
  • Actualización automática — Task Scheduler de Windows ejecuta el scraper diariamente a las 6:00 AM
  • Standings EN VIVO — La tabla de posiciones se consulta directamente a ESPN cada vez que se abre la página

📐 Modelos Matemáticos

Distribución de Poisson — Modela la probabilidad de que ocurran k goles dado un promedio λ. Se genera una matriz 7×7 (0-6 goles por equipo) para calcular todas las probabilidades de marcador.

Normal CDF (función de distribución acumulada) — Para métricas como tarjetas y corners, calcula P(X > línea) usando la media ponderada y desviación estándar.

Distribución Exponencial — Estima el minuto del primer gol basándose en la tasa combinada de gol por minuto, ajustada por presión ofensiva (tiros a puerta) y solidez defensiva (clean sheets).
